Portfolio Website - Anisha Catherine Jose

About the Project

This is a personal portfolio website for Anisha Catherine Jose, a web developer from India. The website showcases Anisha's skills, projects, and experience, providing visitors with an insight into her work.

Features

Responsive Design: The website is fully responsive, ensuring a seamless experience across devices.

Navigation Bar: Smooth navigation with links to different sections (Home, About, Portfolio, Skills, Contact, Blog).

Hero Section: A visually appealing introduction with key details about Anisha.

About Section: Detailed background, skills, and interests.

Portfolio: Displays various projects categorized into Web Development, AI Projects, and Others.

Skills: Progress bars showcasing proficiency in different technologies.

Contact Form: Allows visitors to reach out via email.

Blog Section: Displays latest blogs related to web development and technology.

Download CV Button: Quick access to Anisha's resume.

Technologies Used

Frontend:

HTML5

CSS3

JavaScript

Tailwind CSS

APIs & Libraries:

Google Fonts

OpenCV & TensorFlow (for AI projects)

Project Showcase

Some of the notable projects featured in this portfolio:

Weather WebApp - A real-time weather application using JavaScript and OpenWeatherMap API.

Wrong Way Vehicle Detection - AI-powered computer vision project for detecting wrong-way drivers.

Culprit Face Identification System - Facial recognition project utilizing deep learning.

Installation & Usage

To run this project locally:

Clone the repository:

git clone https://github.com/anisha11111/portfolio.git

Navigate to the project folder:

cd portfolio

Open index.html in a browser.
